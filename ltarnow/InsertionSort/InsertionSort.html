The “insertion sort” sorting algorithm is a simple sorting algorithm capable of organizing an array in sorted order. The insertion sort algorithm has the worst-case running time of O(n2). The insertion sort algorithm iterates through an input array and removes one element per iteration, temporarily stores the element, and places the temporary element in the lowest position in the array possible. It decides where to place the temporary element by comparing the temporary element with every other element in the array, and place the temporary element in the array positioned below an element larger than it, but higher than the elements below it. For each element A[ i ], if A[ i ] > A[ i + 1], swap the elements until A[ i ] ≤ A[ i + 1 ]. A simple example of this is demonstrated in Java:

 

public static void sort( int arr[] )

{

int N = arr.length;

int i, j, temp;

for (i = 1; i< N; i++)

{

j = i;

temp = arr[i];

while (j > 0 && temp < arr[j-1])

{

arr[j] = arr[j-1];

j = j-1;

}

arr[j] = temp;

}

}
