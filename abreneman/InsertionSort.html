Insertion Sort Technical Writing

 

Amanda Breneman

 

Insertion sort creates a sorted list by finding the correct place for each element one at a time – this example will sort the elements in ascending order. The code that implements this sort is comprised of two loops: one loop to iterate through each element in the list and a second loop to move an element to its correct place, if it is not already there.

 

int temp;

for (int i = 1; i < input.length; i++) {

     for(int j = i ; j > 0 ; j--) {

         if(input[j] < input[j-1]) {

             temp = input[j];

             input[j] = input[j-1];

             input[j-1] = temp;

          }

     }

}

 

The sort starts with the second element of the list and compares it to the first element. If the second element is smaller, then it will get moved to the first position in the list and the first element will shift right one place. The third element is then compared to the second element and if it is smaller, it will get shifted left until it is in the correct place. This pattern of evaluating an element and shifting it left until the element is in sorted order is followed until the last element in the list is placed in its correct location.
