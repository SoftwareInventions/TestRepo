Insertion Sort Technical Writing

 

Amanda Breneman

 

Insertion sort creates a sorted list by finding the correct place for each element one-by-one – this example will sort the elements in ascending order. The sort starts at the second element of the list and compares it to the first element. If the second element is smaller, then it will get moved to the first position in the list and the first element will shift right one place. Then the third element is compared to the second element and if it is smaller, it will get shifted left until it is in the correct place. This pattern of evaluating the element and shifting it left until it is in sorted order is followed until the last element in the list is placed in its correct location.
