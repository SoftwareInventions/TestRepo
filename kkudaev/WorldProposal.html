World Description

  • The World has random boundaries

Implementation: 2 random numbers for x and y axis length.

  • The World has a random landscape (lakes and mountains)

Implementation: A random 2D height map array will be generated using midpoint displacement algorithm. Either Water, Ore, or Stone will be placed in a cell based on the cell’s height.

  • Trees and Cows are randomly placed in the world.

Implementation: A random number will be generated to decide if a Cow, a Tree, or nothing should be placed in each cell (except for the cells that already have either Water, Ore, or Stone in them).
