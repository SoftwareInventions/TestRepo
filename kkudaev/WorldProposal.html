World Description

  • The World has random boundaries

Implementation: 2 random numbers for x and y axis length.

  • The World has a random landscape (lakes, mountains, etc.)

Implementation: A random 2D height map array will be generated using midpoint displacement algorithm. Either Water, Stone, Tree, or Ore will be placed in a cell based on cell’s height.
