Assume we need to sort the elements of an array of alphabetic characters in descending order, ie. from ‘A’ to ‘Z’. We assume the array begins with an element at index 0, a common practice in computer science applications. Using the Insertion Sort algorithm, we designate the element at index 0 in the array sorted; a single element array is already sorted. The element at index 0 is ‘Y’. At index 1, we find the letter ‘L’. We swap ‘L’ for ‘Y’ at index 1; index 0 has the element ‘L’ as a result of the swap. Now index 0 and index 1 are in the sorted part of the array.  We continue swapping elements, and remembering the largest index of the sorted part of the array, until the largest index sorted reaches the length of the array.  

--Insertion Sort Facts--

*Insertion sort functions optimally on lists with lengths between 5 and 10 elements.

*Insertion sort performs poorly on lists with a reverse order.
