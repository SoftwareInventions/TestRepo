Insertion Sort

Nicole Gerber

An insertion sort is an algorithm that sorts a collection of numbers (usually an array) by using a pair of nested loops. The outer loop starts at the second number in the array and iterates forward, while the inner loop starts at the current position of the outer loop in the array and iterates backward. As the inner loop traverses backward through the array, the code inside of the inner loop compares the number at the current position to the number at the position behind it, and swaps the numbers if they are not in order.

for (i = 1; i < array.length; i++) {
   for (j = i; j > 0; j--) {
      if (array[j] < array[j - 1])
         swap(array[j], array[j - 1]);
   }
}

By this method, the array is sorted as if each progressive number is “inserted” into its correct place in the array.
